# Simple Interest Calculator

A simple tool to calculate the interest earned or paid on a principal amount over a specific period at a fixed interest rate.

## Formula
The formula used to calculate simple interest is:
**I = P * r * t**

Where:
*   **P** = Principal amount
*   **r** = Annual interest rate (in decimal form)
*   **t** = Time period (in years)

## How to use
1. Input the principal amount.
2. Input the annual interest rate.
3. Input the time period in years.
4. The calculator will output the total interest earned.
